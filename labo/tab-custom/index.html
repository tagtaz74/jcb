タブ表示数カスタムサンプル

タブごとに表示/非表示を個別に切り替えられる構成のテストページです。
下の操作パネルのチェックボックスで、表示するタブの組み合わせを自由に変更できます。

仕組み

  • 各タブのボタン／パネルには共通の data-tab 値（card-sw / card-gold / card-platinum）が付いており、チェックボックスのON/OFFに応じて対応するボタン・パネルの data-hidden 属性を true / false に切り替えます。
  • data-hidden="true" のボタン／パネルはCSSで非表示になります（data-hidden はタブごとの独立したフラグなので、任意の組み合わせで表示できます）。
  • 表示中のタブ数は毎回JSで数え直し、.tabs の data-visible-count 属性に反映します。これが1の場合、ボタンはクリック不可・見出し風の見た目にして不自然にならないようにしています。
  • .tabs--hide-single-nav クラスを付与すると、表示タブが1つの時にナビ自体（.tab-nav）を非表示にできます。
  • 全てのタブを非表示にはできないよう、最後の1つが外れる操作はJSで無効化しています。
  • 実運用ではチェックボックスの代わりに、各タブの data-hidden 属性を手動で更新して出し分ける想定です。
